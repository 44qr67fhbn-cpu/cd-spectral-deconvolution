# cd-spectral-deconvolution
CD spectral deconvolution (Python)

This repository contains the Python script used to document the numerical procedure for CD spectral analysis in our manuscript.

Files
- cd_deconvolution.py: reproducible estimator that maps mean ellipticity values at 208/217/195 nm to relative secondary-structure fractions (alpha-helix, beta-sheet, turn, random coil).

How to run
1) Install Python 3
2) Run:
   python cd_deconvolution.py

Input
Replace the example mean ellipticity values (208, 217, 195 nm) with the values used in the manuscript.
